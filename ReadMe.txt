Instructions:

Open the Backend folder in Visual Studio Code (VSC) and run it in Python debug mode. It should run on port 5000.
Open the Frontend folder in VSC and run ng serve in the terminal.
Ensure the environments/environment.ts file has the apiService URL set to http://localhost:5000/.
Features:
A. Clicking on any polygon (zone) will display the zone's properties in the configuration section with an option to delete the zone.
B. By clicking the 'Add New Zone' button, you can use your mouse to click and define the points for a new zone, then save the zone.
C. A 1-second delay has been added when retrieving the zones to display the loading-spinner component.
